# Contributing to PF2E Weapon Builder

Thank you for your interest in contributing!

## How to Contribute

1. Fork the repository and clone your fork.
2. Create a new branch for your feature or bugfix.
3. Make your changes and add tests as needed.
4. Commit your changes with clear, descriptive messages.
5. Push your branch to GitHub.
6. Open a pull request against the main branch.

## Code Style
- Use consistent formatting and lint your code (`npm run lint`).
- Write unit tests for new features or bugfixes.
- Keep commits focused and atomic.

## Reporting Issues
- Use the bug report template in the Issues tab.
- Provide clear steps to reproduce, screenshots, and environment details.

## Feature Requests
- Use the feature request template in the Issues tab.
- Describe your idea and why it would be useful.

## Community
- Be respectful and constructive in all interactions.
- All contributions are welcome, regardless of experience level.
